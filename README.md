## ⚙️ How to Run This Project

1. Clone the repo:
   git clone https://github.com/Harris1250/muhammad-portfolio.git
   cd muhammad-portfolio

2. Open `index.html` in your browser.

Everything runs client-side with HTML/CSS/JS only.
